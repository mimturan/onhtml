## ✨ Why onHTML?

- **All-in-one** — HTML / CSS / JS playground **plus** 60+ developer tools in a single tab
- **Private** — code runs entirely in your browser, nothing is uploaded
- **Free forever** — no signup, no ads, no upsell
- **Multilingual** — EN · ES · ZH · HI · FR · TR
- **Modern UX** — dark mode, autosave, shareable snippets, cheatsheet
